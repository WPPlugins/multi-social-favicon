This plugin use social profile image to creating a favicon from social site like G+, MSN Live, Twitter, Friendfeed, Facebook and Gravatar.

== Description ==

Multi social favicon is a wordpress plugin instantly make a custom favicons with your social profile image for your website or blog. It’s allowed blogger to choose their profile image to creating a  website favicon from major social web site like [Google+](https://plus.google.com/), MSN Live, [Twitter](http://twitter.com/), [Friendfeed](http://friendfeed.com/), [Facebook](http://facebook.com.com/) and [Gravatar](http://gravatar.com/).

Just activate the plugin and go to Settings to configure the plugin with your social account identity. This plugin assist in building the required images included Apple touch icon and stored in the `root` of a website or in the case of WordPress, the same location as the `uploads folder`.

Questions? Please visit [my blog](http://patrick.bloggles.info/2011/08/31/new-wordpress-plugin-multi-social-favicon/)

== Installation ==
Support Wordpress and Wordpress Multisite

= Standard Multisite Plugins installation. =

1. Upload `multi-social-favicon.php` to the `/wp-content/mu-plugins/` directory

= Standard Plugins installation. =

1. Upload `multi-social-favicon.php` to the `/wp-content/plugins/` directory.

2. Activate the plugin through the `Plugins` menu in WordPress

3. Enter your social account identity in `General Settings`

= How do I disable the Multi Social Favicon? =

You can disabled the web site icon option in General Settings. Then the favicon will turn off.

All icons or images file will be removed if plugin deactivated.
